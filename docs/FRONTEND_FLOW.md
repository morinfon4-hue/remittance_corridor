# Frontend Flow

This document explains the frontend flow for remittance_corridor.

## Dashboard Areas

<pre>
Hero section
Contract information card
Corridor metrics
Wallet connection panel
Remittance action form
Prepared action preview
Recent transfer list
</pre>

## User Flow

<pre>
Open frontend
Review contract ID
Connect Freighter wallet
Enter transfer details
Prepare contract action
Review action preview
Use status panel for feedback
</pre>

## Design Goal

The frontend is designed as a clean dashboard instead of a basic form-only page.
