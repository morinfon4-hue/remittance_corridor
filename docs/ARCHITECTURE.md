# remittance_corridor Architecture

remittance_corridor is a Stellar Level 3 dApp for a cross-border remittance corridor on Stellar testnet.

## Main Parts

<pre>
Soroban smart contract
React frontend
Freighter wallet flow
PowerShell scripts
GitHub Actions CI
Documentation
Deployment records
</pre>

## Contract

<pre>
contracts/remittance_corridor
</pre>

The contract models a remittance transfer lifecycle between a sender, recipient, payout partner, and admin.

## Frontend

<pre>
frontend
</pre>

The frontend is used to present the corridor dashboard, wallet status, contract ID, and remittance action panels.
