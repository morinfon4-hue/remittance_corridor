# Security and Privacy Notes

remittance_corridor is a testnet demo project.

## Privacy Rule

<pre>
Do not store private user identity data
Do not store bank account details
Do not store private payout records
Use testnet-only values during development
</pre>

## Contract Safety Notes

<pre>
Validate transfer status before updates
Restrict admin-only actions
Keep dispute and release flows explicit
Keep deployment records separate from private keys
</pre>

## Local Safety Notes

<pre>
Do not commit .env files
Do not commit wallet secrets
Do not commit generated build folders
</pre>
